Pengalaman Saya Ketika Melihat JKT48 Di Solo

Ditulis oleh Fairuz Aqila Islami,tanggal 14,juli,2023

Pada hari itu saya diajak oleh temen saya untuk mengikuti SUMMERTOUR JKT48 di solo. Tetapi awalnya saya menolak ajakan temen saya karena, JKT48 menurut saya biasa saja di karenakan di dalam pikiran saat itu saya JKT48 hanyalah sekelompok idol yg hanya bernyanyi dana menari,dan setelah saya pikir2 saya menerima ajakan temen saya untuk mengikuti SUMMERTOUR JKT48 di (SOLO)The park mall

dan di pikiran saya saya mengikutievent tersebut hanya untuk menghilangkan rasa gabut/steres .ketika sampai di The park mall SOLO sekitar jam 11.00 wib di sana sudah ramai pengunjung .untuk Melihat konser JKT48 di hati saya berbicara 'rame bener dah emanag nya JKT48 se bagus itu ya?' tanya saya yang masih heran.dan setelah menunggu 4 jam sekitar jam 02.05 akhinya para member JKT48 yg terdiri dari Adel,Zee,Ashel,Shani,Muthe,DLL di sana mereka sudah berada di panggung.saya kira bakal langsung konser/mulai pertunjukannnya,ternyata masih ada acara meat and great dari para member ke menonton dan bayar kisaran 50k/10detik dan saatitu saya mulai bosan pada acara meet and great awalnya saya ingin pulang cepat,tetapi temen saya memaksakan saya untuk melihat JKT48 hingga akhir dan kemudian saya menyetujui itu.

Dan malam pun tiba sekitar jam 08.00 acara konser JKT48 akan di mulai, saat itu ekspetasi saya konser JKT48 seperti konser pada umumnya, tetapi setalah di mulai konser tersebut saya melihat perfom JKT48 ketika saya lihat entah mengapa di konser tersebut saya sangat berbahagia dan gembira. di situ merka menyayikan lagu yang belum pernah saya dengar dan penonton pun yg melihat perfomdari member JKT48 sangat bahagia dan di situlah saya menyuakai JKT48 pada 14,juli,2023 saya akan selalu ingat tanggal itu bahwasanya saya menyukai JKT48 karena bisa menonton meraka tampil.
